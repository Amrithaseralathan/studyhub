Implementation of Simple Calculator.

Here we will see program to create a simple calculator which can perform basic arithmetic operations like addition, subtraction, multiplication or division depending upon the user input.

Source code:

public static void main(String[] args){

// stores two numbers

double num1, num2;

// Take input from the user

Scanner sc = new Scanner(System.in);

System.out.println("Enter the numbers");

// take the inputs

num1 = sc.nextDouble();

num2 = sc.nextDouble();

char op = sc.next().charAt(0);

double o = 0;

switch (op) {

// case to add two numbers

case '+':

o = num1 + num2;

break;

// case to subtract two numbers

case '-':

o = num1 - num2;

break;

// case to multiply two numbers

case '*':

o = num1 * num2;

break;

// case to divide two numbers

case '/':

o = num1 / num2;

break;

default:

System.out.println("You enter wrong input");

break;

}

System.out.println("The final result:");

System.out.println();

// print the final result

System.out.println(num1 + " " + op + " " + num2 + " = " + o);

}

}

Output:

Enter the numbers:

2

2

Enter the operator (+,-,*,/)

The final result:

2.0 + 2.0 = 4.0
